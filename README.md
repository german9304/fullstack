# Fullstack Application


Start application
```docker
 docker-compose start
```
Stop application
```docker
 docker-compose stop
```


## Tools used

Devops 

* [docker](https://www.docker.com/)


Back end

* [mysql](https://www.mysql.com/)
* [golang](https://golang.org/)
* [graphql](https://graphql.org/)

Front end

* [React](https://reactjs.org/)
